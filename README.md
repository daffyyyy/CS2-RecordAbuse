# CS2-RecordAbuse

### Description
Allows admins to record demo with command

### Requirments
[CounterStrikeSharp](https://github.com/roflmuffin/CounterStrikeSharp/) **tested on v126**

### Commands
```
css_record <start/stop/status>
```
